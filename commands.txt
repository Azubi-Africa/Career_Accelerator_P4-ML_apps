# Setup virtual environment

## One-line Setup
python3 -m venv venv | venv\Scripts\activate | python3 -m pip install --upgrade pip | pip3 install -qr requirements.txt |  

## ManualSetup
### Create the venv
python3 -m venv venv

### Activate the venv
source venv/bin/activate        #this line will work in linux
venv\Scripts\activate           # this it the code  for windows.

### Deactivate the venv
deactivate


### Install required packages
python -m pip install --upgrade pip               # update pip first before to use it, to avoid warning
python -m pip install -qr requirements.txt
